Pirates of the Cannonean

GitHub

Overview

This is a website displaying data about maritime pirate attacks. This Website uses the Python Flask framework to display information regarding location and frequency of pirate attacks around the world. This data can be explored within the explore page in map form, and the charts page in charts describing number of attacks in different regions.

How to Use

To run this project, open a terminal in whatever directory you download these files to and run python3 project.py from the GitHub link.
